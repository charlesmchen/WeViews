We create and configure the WePanel with the first statement. The second statement creates and configures a new layer which contains three WeButtons.
WeViews offers subclasses of many popular UIViews. In this case, WeButton is a subview of UIButton.
Like many WeViews, WePanel and WeButton offer a variety of factory methods. WeViews factory methods are named "create" and return a new, autoreleased instance.
WeViews configuration methods often return a reference to the instance so that invocations can be chained. This makes for more readable code.
The first statement can be read "Create a new WePanel with X background color." The second statement can be read "Add a new horizontal layer to WePanel with buttons A, B and C that has right and bottom alignment and 10 point margins."

Properties: Margins & Alignment

We can set the margins and alignment properties of the horizontal layer. In this case, we align Bottom and Right with 10 point margins.
