Declared in WeCustomLabel.h
WeCustomLabel.m

Overview

Alternative to UILabel that includes support for CSS-style margins, custom line spacing. Imitates UILabel/WeCustomLabel as closely as possible.

Text wrap is always enabled.

Only use WeCustomLabel if you need it’s extra functionality. Otherwise use UILabel or WeCustomLabel.

See WeCustomLabel.
