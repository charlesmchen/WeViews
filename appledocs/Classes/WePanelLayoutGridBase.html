Overview

Base class for grid layouts.

Has either columnCount or rowCount. The other is calculated based on the number of items.

uniformGrid

If YES, all columns have width of widest column and all rows have height of tallest row.

@property (nonatomic, assign) BOOL uniformGrid

Discussion

If YES, all columns have width of widest column and all rows have height of tallest row.

This property isn’t honored by all subclasses of this class.
