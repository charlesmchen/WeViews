Inherits from WeBaseObject : NSObject
Declared in WePanelLayer.h
WePanelLayer.m

Overview

An WePanel’s content are usually (but don’t have to be) organized in layers. Each layer has its own layout state.

WePanel properties are usually set with “with…” methods that can be chained.
