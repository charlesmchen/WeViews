findUIFont:fontSize:

Find a UIFont by “name”.

+ (UIFont *)findUIFont:(NSString *)fontName fontSize:(CGFloat)fontSize

Parameters

fontName

The “name” of the font.

fontSize

The point size.

Return Value

The UIFont or nil if it cannot be found.

Discussion

Find a UIFont by “name”.

Fonts have many names: the “display” name, the “postscript” name, the file name, etc. Also, different versions of iOS handling font names slightly differently. This method papers over these subtleties.
