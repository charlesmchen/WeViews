WePanel Class Reference

Inherits from WeView : UIControl
Declared in WePanel.h
WePanel.m

Overview

A container for UIViews can automatically layout its contents using a variety of layout strategies.

Subviews are added in layers (WePanelLayer).

See: WePanelLayer, WePanelLayout.
